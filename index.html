Node Browser

A lightweight web browser built with Node.js.

About

Node Browser is a lightweight browser powered by Node.js. It is designed to be fast, simple, and easy to use while keeping resource usage as low as possible.

  • ⚡ Lightweight and fast
  • 🟢 Powered by Node.js
  • 🖥️ Clean and modern interface
  • 📦 Easy to install
  • 🔄 Regular updates

Builds are available from the MiniNode website.
